- Moving, scaling, rotating:
    - G (+x +y +z) (+x/y/z +<number>)
    - S (+x +y +z) (+x/y/z +<number>)
    - R (+x +y +z) (+x/y/z +<number>)

- Viewing angles (preferences > input > emulate numpad):
    - y view: 1
    - x view: 3
    - z view: 7
    - camera view: 0
    - set camera view: opt+cmd+0
    - adjust camera view (chickenhead): shift+`

- Viewing modes:
    - Z (hold) + mouse or number

- Other:
    - Delete: x + d
    - Add: shift + a
    - Duplicate: shift + d
